# Security Policy

## Reporting a vulnerability

Email suspected security vulnerabilities to
[hello@reepolee.com](mailto:hello@reepolee.com). Do not open a public
issue before disclosure has been coordinated.

Include the affected version, a minimal reproduction or proof of concept, the
expected impact, and any mitigation you have already identified. Reepolee Labs
will acknowledge the report and coordinate next steps directly with you.

## Supported versions

Reepolee is currently in beta. Security fixes are made on the current
development line; there is not yet a published long-term-support version.
